# Rota Montenegro 🚍

Web app para consulta de horários de transporte coletivo (ônibus urbano, intermunicipal e comunitário) das localidades do interior de Montenegro, RS.

## Motivação

Quem mora nos distritos mais afastados de Montenegro depende de linhas de ônibus para resolver questões básicas no centro da cidade. Hoje, esses horários estão espalhados em PDFs e sites pouco intuitivos. O Rota Montenegro centraliza essa informação num só lugar, com consulta rápida e sem necessidade de cadastro.

Projeto de estudo, desenvolvido para praticar Python/Django com um caso real, como parte da minha formação em Análise e Desenvolvimento de Sistemas.

## Status

Em desenvolvimento. A categoria **Urbano** já está disponível, com dados reais extraídos de PDF público e migrados para o banco via Django ORM. Ainda faltam as categorias **Intermunicipal**, **Interiorano** e **Seletivo**.

Para o futuro, a ideia é evoluir para um app mobile, com mais funcionalidades além da consulta.

## Stack

Python, Django, SQLite, Django Templates, Bootstrap 5.
